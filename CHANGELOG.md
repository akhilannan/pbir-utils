### New Features

#### CLI Support
The project now includes a Command Line Interface (CLI) to easily access all utility functions without writing Python scripts.

**Available Commands:**
- `sanitize`: Sanitize a Power BI report by removing unused or unwanted components.
- `extract-metadata`: Export attribute metadata from PBIR to CSV.
- `visualize`: Display report wireframes using Dash and Plotly.
- `batch-update`: Batch update attributes in PBIR project.
- `disable-interactions`: Disable visual interactions between visuals.
- `remove-measures`: Remove report-level measures.
- `measure-dependencies`: Generate a dependency tree for measures.
- `update-filters`: Update report-level filters.
- `sort-filters`: Sort report-level filter pane items.
